Kapcontrol Blog Migration

October 2025 Next.js Supabase

Overview

The Kapcontrol Blog project involved migrating an existing blog platform from a traditional Express.js backend to a modern, serverless architecture using Next.js and Supabase. This migration improved scalability, reduced server costs, and simplified the deployment pipeline.

Key Contributions

  • Removed Express.js backend and replaced with Supabase database, auth, and storage
  • Implemented Row Level Security (RLS) for data protection
  • Built responsive UI with Next.js App Router and Tailwind CSS
  • Integrated GitHub OAuth via Supabase for authentication
  • Set up CI/CD pipeline with Vercel deployment
